<h2><a href=https://www.codewars.com/kata/584c702256c98fc0a0000043/train/javascript target="_blank">Fizz Buzz - Without ifs</a></h2><h3>7 kyu</h3><p><em>I know that there are quite a few fizz-buzz problems already out there but this one does have a twist <a href="http://www.w3schools.com/js/js_comparisons.asp" data-turbolinks="false" target="_blank">(you may find this useful)</a></em></p><h2 id="task">Task</h2><p>You need to implement a function that applies the standard fizz buzz test:</p><ul><li>If a number is divisible by 3, return <code>Fizz</code></li><li>If it is divisible by 5, return <code>Buzz</code></li><li>If it is divisible by both 3 and 5, return <code>FizzBuzz</code></li><li>If it is not divisible by 3 or 5, return the number itself</li></ul><p>However, when making the function, <strong>you cannot</strong> use <code>if</code> (meaning <code>else</code> is also not an option).</p><h2 id="input">Input</h2><ul><li>A number will always be provided</li><li>The number provided will always be positive and an integer</li></ul><h2 id="examples">Examples</h2><ul><li><code>6</code> should return <code>Fizz</code></li><li><code>10</code> should return <code>Buzz</code></li><li><code>15</code> should return <code>FizzBuzz</code></li><li><code>17</code> should return <code>17</code></li></ul>
